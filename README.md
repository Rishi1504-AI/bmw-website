# BMW | Cinematic Luxury Auto Showcase

A high-fidelity, interactive web experience built to showcase the pinnacle of BMW engineering and design. This project leverages modern web technologies to create a cinematic, editorial-style journey through the BMW lineup.

![BMW Showcase](https://img.shields.io/badge/Experience-Cinematic-blue)
![Tech Stack](https://img.shields.io/badge/Tech-React%20%7C%20GSAP%20%7C%20Three.js-black)

## 🌟 Key Features

### 1. **Cinematic Hero Experience**
A fullscreen, optimized video-driven introduction with smooth entrance animations and minimalist typography, establishing the brand's premium identity from the first second.

### 2. **Interactive 3D Model Showcase**
A dedicated "All Models" laboratory where users can explore high-quality 3D car models.
*   **Categories:** SUV, Sedan, Coupe, and Convertible.
*   **Interactions:** 360° rotation, dynamic lighting, and real-time spec overlays.
*   **Infinite Loop:** Seamless browsing through the entire model lineup.

### 3. **Stacked Scroll Narrative**
A custom-engineered scrolling experience on the home page where car models stack and reveal themselves with cinematic depth and GSAP-powered transitions.

### 4. **Innovation & Technology Hub**
A specialized section detailing BMW's technical advancements, featuring a modern interactive grid and scroll-triggered content reveals.

### 5. **Premium UI/UX**
*   **Custom Luxury Cursor:** A dual-layered interactive cursor that reacts to navigation elements.
*   **Performance Counters:** Real-time counting animations for 0-100 speed, horsepower, and top speed stats.
*   **Glassmorphism Design:** Subtle blurs and semi-transparent layers for a modern, high-end aesthetic.

## 🛠️ Tech Stack

*   **Frontend:** React.js, Vanilla JavaScript, HTML5, CSS3
*   **Animations:** GSAP (GreenSock Animation Platform) with ScrollTrigger
*   **3D Rendering:** Google Model-Viewer (WebGL-based)
*   **Styling:** Tailwind CSS & Vanilla CSS
*   **Build Tool:** Vite

## 🚀 Getting Started

### Prerequisites
*   Node.js (v16.x or higher)
*   npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/bmw-website.git
   ```

2. Navigate to the project directory:
   ```bash
   cd bmw-website
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Build for production:
   ```bash
   npm run build
   ```

## 📐 Design Philosophy
The project follows a **"Digital Grandeur"** philosophy:
*   **Minimalism:** "Less is more." High-contrast layouts with significant negative space.
*   **Car is King:** UI elements are secondary; the car models and visuals are always the primary focus.
*   **Smoothness over Speed:** Transitions are intentional and cinematic, prioritizing a premium feel over rapid-fire content delivery.
